# LEGO Database SQL Analysis 
 
SQL analysis of Rebrickable LEGO database executed in Python/Jupyter environment. 
 
## Dataset 
 
12 CSV files from Rebrickable.com (Dec 30, 2025): 
- themes.csv 
- colors.csv 
- part_categories.csv 
- parts.csv 
- part_relationships.csv 
- elements.csv 
- sets.csv 
- minifigs.csv 
- inventories.csv 
- inventory_parts.csv 
- inventory_sets.csv 
- inventory_minifigs.csv 
 
## Tech Stack 
 
- PostgreSQL 
- Python 
- Jupyter Notebook 
- SQL (CTEs, JOINs, Aggregations, Subqueries) 
 
## Analysis 
 
- Production trends (177 sets/year average) 
- Color patterns (Black most popular: 115,085 parts) 
- Set complexity evolution (peak 261 parts in 2017) 
- Theme popularity (Supplemental: 496 sets) 
- Part rarity analysis
